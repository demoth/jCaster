jCaster
=======

Caster is a turn based multiplayer rogue like game. jCaster is a standalone 3d java client.
Libs are currently used (place them in the lib directory):
  * jmonkeyengine 3
  * JSON libray from json.org
  * slf4j - simple

Jars needed (lib/):
  * jMonkeyEngine3.jar  
  * JSON.jar  
  * slf4j-api-1.6.6.jar  
  * slf4j-simple-1.6.6.jar
